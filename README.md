## Instalaci√≥n

1. Clona el repositorio:
   ```bash
   git clone https://github.com/raul-gabriel/sistema_ventas.git
   ```
2. entrar al carpeta:
   ```bash
   cd sistema_ventas
   ```
3. darle permisos:
   ```bash
   chmod +x install.sh
   ```
4. ejecutar:
   ```bash
   ./install.sh
   ```
5. descomprimir:
   ```bash
   7z x -p[password] -y sistema_ventas.zip
   ```
6. ejecutar:
   ```bash
   cd sistema_ventas
   npm i
   ```
